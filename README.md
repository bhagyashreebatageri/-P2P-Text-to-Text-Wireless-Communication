# P2P Text Based Wireless Communication With Zero Internet

## Overview

This project implements a Peer-to-Peer (P2P) wireless communication system using ESP32 and LoRa SX1278 modules. The system enables users to exchange text messages without requiring internet connectivity, cellular networks, or centralized servers.

## Features

* Internet-free communication
* Long-range LoRa messaging
* ESP32-based Wi-Fi Access Point
* Web-based messaging interface
* OLED message display
* Low power consumption
* Secure communication

## Hardware Used

* ESP32 Development Board
* LoRa SX1278 (RA-02) Module
* OLED Display (SSD1306)
* Jumper Wires
* Power Supply

## Software Used

* Arduino IDE
* Embedded C/C++
* LoRa Library
* WiFi Library
* Adafruit SSD1306 Library

## Applications

* Smart Campus Communication
* Disaster Management
* Emergency Communication
* Rural Area Messaging
* Industrial Communication
