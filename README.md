# P2P Text Based Wireless Communication With Zero Internet

## 📖 Project Overview

The P2P Text Based Wireless Communication With Zero Internet project is designed to enable direct text communication between users without relying on internet connectivity, cellular networks, or centralized servers. The system uses ESP32 microcontrollers and LoRa SX1278 modules to establish secure, long-range wireless communication between multiple nodes.

Users can connect to the ESP32 through a Wi-Fi Access Point and access a web-based messaging interface using a smartphone, tablet, or laptop. Messages are transmitted through LoRa technology and displayed on an OLED screen at the receiving node.

## 🎯 Objectives

* Develop a Peer-to-Peer (P2P) communication system without internet connectivity.
* Enable long-range wireless text communication using LoRa technology.
* Create a web-based messaging platform accessible through Wi-Fi.
* Display received messages on OLED displays.
* Provide secure and reliable communication between nodes.
* Build a low-cost and energy-efficient communication solution.

## ⚙ Working Principle

1. ESP32 creates a Wi-Fi Access Point.
2. Users connect using smartphones, tablets, or laptops.
3. A web-based dashboard is accessed through a browser.
4. Users compose and send messages.
5. ESP32 converts messages into LoRa packets.
6. LoRa SX1278 transmits the packet wirelessly.
7. Destination node receives the packet.
8. Received messages are displayed on the OLED display and web dashboard.
